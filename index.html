    about this project & copyright

    This page was built as a personal creative project using HTML, CSS and JavaScript. Music is streamed exclusively through official Spotify embed players — the same embeds Spotify provides for use on any website, blog or portfolio. No audio files are hosted, downloaded or redistributed here.

    ✅ What this site does

    Displays song names, artists and album artwork — all publicly available metadata. Uses Spotify's official embed widget to play previews directly from Spotify's servers.

    🚫 What this site does NOT do

    No audio is hosted or downloaded. No copyrighted content is reproduced. No DRM is bypassed. The embed is identical to pasting a YouTube video on a webpage.

    📄 Legal basis

    Spotify embeds are explicitly permitted under Spotify's Embed API terms. Album art is served directly from Spotify's CDN via their API.

    This project is not affiliated with or endorsed by Spotify AB. All music rights belong to their respective artists and labels. Built with love for portfolio purposes. 🌸
